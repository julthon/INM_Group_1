How does the Javascript Syntax work?

Case sensitivity

JavaScript is case sensitive. That means that the variable names "house" and "House" are different and can be used as two seperate variables.

Whitespace and semicolons

JavaScript uses semicolons (;) to mark the end of a line of code. However, some interpreters do not need this at every end of line. For instance if the last line in a code block is a return statement, it will be executed even without a semicolon: 

    function x() {
      a = 1;
      return a // no ";" needed
    }

Comments

The syntax for comments in JavaScript is the same as in C/C++. Single line comments can be made by inserting two slashes (//) and everything that comes after this in the same line will be interpreted as a comment.

Multiline comments can be made by putting /* before and */ after the comment. This can also be used to only comment out parts of a line.

JavaScript Literals

Numbers can be written either with or without a decimal point: 

    num1 = 3.50
    num2 = 101

Strings can be written by using single or double quotes: 

    str1 = "Test"
    str2 = 'Test'

JavaScript Operators

JavaScript uses the equal sign (=) an assignment operator: 

    var x = 5;
    var y = 6;

JavaScript uses the standard arithmetic operators ( + - * / ) and brackets () to express calculations: 

    (1 + 2) * 3
